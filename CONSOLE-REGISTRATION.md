## 상세 설명 (복붙용, ≤500자, 이모지 X)

```
국내와 미국에 상장된 ETF를 종가와 기준가(NAV) 기준으로 살펴보는 정보 앱이에요.

홈에서는 국내와 미국을 나눠 거래량, 상승, 하락 순으로 ETF를 모아 볼 수 있어요. 종목을 누르면 상세 화면에서 더 자세히 확인할 수 있어요.

국내 ETF 상세
- 종가와 기준가 기준 괴리율, 추적오차
- 총보수, 분배수익률, 시가총액, 과세유형
- 기간별 수익률
- 주요 구성종목과 비중, 섹터 비중

미국 ETF 상세
- 달러 기준 시세와 등락률
- 분배수익률, 주당 분배금, 시가총액
- 주요 구성종목과 섹터 비중

검색에서는 ETF 이름뿐 아니라 특정 종목명으로 그 종목을 담고 있는 ETF를 거꾸로 찾을 수 있어요. 테마와 유형 칩으로 빠르게 좁힐 수도 있어요.

모든 수치는 외부에서 수집한 참고용 데이터이며, 투자 자문이 아니에요.
```

```
국내와 미국 ETF를 종가와 기준가 기준으로 한눈에 모아 봐요. 홈에서 국내·미국을 나눠 거래량·상승·하락 순으로 훑고, 종목을 누르면 괴리율·추적오차·총보수·분배수익률과 주요 구성종목, 섹터 비중까지 상세히 확인할 수 있어요. 검색은 ETF 이름은 물론 특정 종목명으로 그 종목을 담은 ETF를 거꾸로 찾아줘요. 모든 수치는 참고용이며 투자 자문이 아니에요.
```

## 앱 내 기능 (비게임 ≥1개 필수)

| 항목 | 입력값 | 메모 |
|---|---|---|
| 한국어 기능 이름 | `오늘의 ETF` | 7자, 명사형 |
| 영어 기능 이름 | `Etf today` | 9자, 첫 글자만 대문자 |
| 이동 URL | `intoss://etf-tracker/` | 홈(인덱스) 진입 |

> ⚠️ **현재 앱은 HashRouter**라 `intoss://etf-tracker/search` 같은 **하위 경로 딥링크는 홈으로 떨어짐**. 하위 기능 딥링크(예: 검색 바로가기)를 등록하려면 BrowserRouter로 전환 필요. 우선 홈 진입 1개로 출시하고, 딥링크 기능은 후속으로.

### 자료 재생성
- 로고/썸네일/가로형: `assets/*.html` 템플릿 → `cd assets && python3 -m http.server 8791` 후 Playwright로 각 viewport(600×600 / 1932×828 / 1504×741) 캡처 → `sips`로 정확 사이즈 리사이즈
- 세로 스크린샷: `cd frontend && npx vite preview --port 4321` → Playwright viewport 636×1048로 홈/상세/검색 캡처

### 디자인 메모
- **로고**: Toss 블루 그라데이션 배경에 흰색 오름차순 바차트(좌하단) + 돋보기(우상단) 대각선 구도. 메타포: 차트=ETF 시세, 돋보기=자세히 들여다보기. 다크모드에서도 흰 글리프라 선명.
- **썸네일/가로형**: 옅은 블루 그라데이션(#EEF4FF→#DCE7FF)으로 로고 톤 통일. 좌측 헤드라인+기능칩, 우측 실제 상세 카드/스크린샷.
